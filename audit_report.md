# Issues Found

## 1. 村里欄位缺失值

-   類型：Missing Value
-   影響：行政區統計與災害分析不完整
-   建議修正：透過地址反查村里，或標記為待確認

## 2. 地址資料不完整（重大）

-   出現「無門牌號碼」
-   僅門牌無路段
-   地址缺失

影響：無法正確定位、GIS 分析失準\
修正方式：建立地址完整性檢查規則、透過經緯度反查補地址

## 3. 室內室外邏輯衝突

-   室內=否 且 室外=否

修正方式：至少一項需為「是」，標記衝突資料人工審查

## 4. 經緯度異常

合理範圍： - 118 \< 經度 \< 123 - 21 \< 緯度 \< 26

超出範圍標記異常

## 5. 災害類別格式不一致

統一格式：水災,震災,土石流

## 6. 電話格式不一致

統一格式：0X-XXXXXXX#分機

------------------------------------------------------------------------

# Overall Assessment

資料結構完整，但缺乏驗證機制。
